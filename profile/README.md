# MegonCSS ⚡

MegonCSS is a lightweight, utility-first CSS framework designed to build modern, responsive, and scalable user interfaces faster with minimal effort.

It focuses on simplicity, performance, and developer experience — just add classes and build.

---

## 🚀 Features

- ⚡ Ultra lightweight and fast
- 🎯 Utility-first class system
- 📱 Fully responsive by default
- 🧩 Easy integration via CDN
- 🔧 No build tools required
- 🎨 Clean and consistent design system
- 🌐 Works with any frontend stack (HTML, PHP, React, etc.)
